# w3cHexSchool 全馬鐵人搜尋器

> 這是一個基於六角學院所推出的 API 而製作的搜尋器，其主要目的是方便搜尋文章以及關注特定使用者文章。

## 使用技術

- Vue Cli4
- Vue Router
- Vue Components
- Boostrap
- jQuery
- SCSS
- Webpack
- ESLint(Airbnb)
- TypeScript
- PWA

## 使用套件

- vue-Axios
- vue-countup-v2
- vue-loading-overlay
- vue-fontawesome
